
# Cœur d'ORA & GrenaPrompt Documentation

Bienvenue dans le dépôt officiel de documentation pour le **Cœur d'ORA** et le **framework GrenaPrompt**.
Ce projet est porté par **TwinsProduction** et consigne l'architecture, l'évolution et les bonnes pratiques
entourant le développement de ce système modulaire d'IA. Publier ce dépôt permet d'établir une trace
horodatée de la paternité de ces concepts.

## Présentation du projet

Le **Cœur d'ORA** est une architecture cognitive composée de quatre noyaux interconnectés :

- **Identité** : définit la personnalité, les valeurs, le ton et les voix employées.
- **Fusion** : gère les styles de communication, les tonalités et le bilinguisme FR/EN.
- **Mémoire** : maintient le contexte conversationnel, les préférences et l'apprentissage adaptatif.
- **Exécution** : orchestre les modes de fonctionnement et les livrables (PDF, DOCX, formats vidéo, etc.).

Le **framework GrenaPrompt** est un langage hybride combinant des structures JSON et du texte naturel.
Il améliore la clarté et la cohérence des instructions destinées aux modèles d'IA. Ce repository
réunit les spécifications, chronologies et guides d'utilisation du système.
